# Owen's Tank Shooting Game

A fun, kid-friendly tank shooting game built with HTML5 Canvas. Designed for children ages 5+.

## How to Play

### Controls

**Keyboard:**
- `←` `→` - Move tank left/right
- `↑` `↓` - Aim barrel up/down
- `SPACE` - Shoot
- `W` - Switch weapon (Cannon/Laser)
- `ESC` - Quit game

**Mobile/iPad:**
- On-screen touch buttons appear automatically on touch devices

### Weapons

| Weapon | Description |
|--------|-------------|
| Cannon | Fires cannonballs affected by gravity - aim high! |
| Laser | Fires straight beams - fast and accurate |

### Enemies

| Enemy | Points | Description |
|-------|--------|-------------|
| Tank | 20 | Ground enemy, moves along terrain |
| Hog | 15 | Fast ground enemy |
| Ghost | 25 | Flying enemy, drops magic fireballs |
| Helicopter | 30 | Flying enemy, drops bombs |

### Difficulty Levels

- **Baby** - Very slow enemies, lots of time to react
- **Easy** - Relaxed gameplay for beginners
- **Medium** - Moderate challenge
- **Hard** - Fast enemies, frequent spawns

### Progressive Difficulty

The game gets harder over time! Watch the "Danger" meter - as it fills up:
- Enemies spawn more frequently
- Enemies move faster

Survive as long as you can!

## Hosting

This is a single-file HTML game. To host it:

1. **GitHub Pages**: Push to a GitHub repository and enable Pages in settings
2. **Local**: Simply open `index.html` in any modern web browser
3. **Any static host**: Upload `index.html` to any web server

## Browser Support

Works on all modern browsers including:
- Chrome, Firefox, Safari, Edge
- Mobile Safari (iOS/iPad)
- Chrome for Android
